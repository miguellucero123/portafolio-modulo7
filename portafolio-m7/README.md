# Carpeta de entrega ZIP (Módulo 7)

Aquí se genera el archivo **`PORTAFOLIO_M7_ENTREGA.zip`** al ejecutar el script de empaquetado desde la raíz del repositorio.

```powershell
.\scripts\empaquetar-portafolio-m7.ps1
```

Detalle de exclusiones y comprobación: [INSTRUCCIONES_EMPAQUETADO.txt](./INSTRUCCIONES_EMPAQUETADO.txt).

Los archivos `*.zip` en esta carpeta están en `.gitignore` para no subir binarios al repositorio; genere el ZIP localmente antes de entregar.
