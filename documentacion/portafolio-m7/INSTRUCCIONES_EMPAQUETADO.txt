================================================================================
  PORTAFOLIO MÓDULO 7 — Archivo ZIP para entrega académica
================================================================================

CONTENIDO ESPERADO DEL ZIP (código fuente)
  - Carpeta src/ (Vue, router, store, vistas, servicios)
  - index.html, vite.config.js, package.json, package-lock.json
  - netlify.toml, public/
  - README.md, RUBRICA_CUMPLIMIENTO.md
  - documentacion/ (guías, proyectos, MODULO7_ENTREGA, etc.)
  - Archivos de configuración del proyecto (.gitignore, etc.)

NO INCLUIR (se regeneran o son locales)
  - node_modules/     → ejecutar "npm install" al descomprimir
  - dist/               → ejecutar "npm run build" si se necesita build
  - .git/               → opcional según indique el docente
  - .netlify/, .vscode/ → entorno local

COPIA SOLO RÚBRICA M7 (sin legacy / clones)

  Desde la raíz del repositorio:

    .\scripts\extraer-rubrica-m7.ps1

  Genera la carpeta documentacion\RUBRICA_M7_CONTENIDO\entrega\ con el proyecto mínimo
  (la copia mínima sigue incluyendo docs\ con DEPLOY y GITHUB para enlaces habituales).
  Luego puede comprimir solo esa carpeta en un ZIP para el docente.

GENERAR EL ZIP (proyecto completo en el repo)

  Desde la raíz del repositorio (carpeta que contiene package.json):

    .\scripts\empaquetar-portafolio-m7.ps1

  El archivo se creará aquí:

    documentacion\portafolio-m7\PORTAFOLIO_M7_ENTREGA.zip

GENERAR EL ZIP A MANO

  Comprima la raíz del proyecto excluyendo las carpetas indicadas arriba,
  o use "Enviar a → Carpeta comprimida" tras eliminar node_modules y dist
  si ocupaban mucho espacio.

VERIFICACIÓN TRAS DESCOMPRIMIR (quien corrige o compañero)

  npm install
  npm run dev
