# Proyecto 3 — Portafolio profesional y difusión digital

Documentación académica para el **Proyecto 3**: aplicación de buenas prácticas en un portafolio de productos, uso de herramientas digitales, identidad profesional y evidencia de mejora continua. Este material se relaciona con el repositorio **ClimaTorre** (`portafolio-modulo7`) como pieza central del portafolio técnico.

**Ámbito del repositorio:** `portafolio-modulo7` (GitHub: `miguellucero123/portafolio-modulo7`).

**Condición de la documentación `proyecto_3/`:** el material de esta carpeta (incluida accesibilidad e inclusión) **complementa** el portafolio; el cumplimiento **obligatorio** sigue alineado a las **rúbricas** del curso ([`RUBRICA_CUMPLIMIENTO.md`](../../RUBRICA_CUMPLIMIENTO.md) para el Módulo 7). Las mejoras opcionales no deben **sustituir** ni **comprometer** esos requisitos.

## Informe único para Word (Parte A + Parte B)

El archivo **[INFORME_PROYECTO_3_COMPLETO.md](./INFORME_PROYECTO_3_COMPLETO.md)** reúne:

- **Parte A:** evidencia del proceso (herramientas, cambios, nuevas secciones) y reflexión personal, alineada al enunciado.
- **Parte B:** el mismo contenido que `00` (contexto **Módulo 7** + **`proyecto_2/`**), `01`, `02` y **`05`** (accesibilidad e inclusión).

Para obtener **.docx:** abrir ese archivo en Microsoft Word y guardar como Word, o ejecutar `pandoc INFORME_PROYECTO_3_COMPLETO.md -o INFORME_PROYECTO_3_COMPLETO.docx` desde esta carpeta (requiere [Pandoc](https://pandoc.org/) instalado).

## Cómo usar esta documentación

1. Leer **`00`** (contexto M7 + proyecto 2), luego **01** y **02**, para tener claro qué evidencia el repositorio antes de redactar el informe.
2. Completar **03** y **04** con información auténtica (sustituir `[COMPLETAR]`); puede citar `RUBRICA_CUMPLIMIENTO.md` y documentos de `proyecto_2/` como respaldo.
3. Revisar **`05`** si debe documentar **accesibilidad e inclusión** como complemento del portafolio (y enlazarlo en herramientas/cambios del informe **03** si aplica).
4. Opcional: generar **Word** desde `INFORME_PROYECTO_3_COMPLETO.md` para la entrega formal.
