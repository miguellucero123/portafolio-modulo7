# 🏔️ ClimaTorre - Portafolio Módulo 6

SPA desarrollada con **Vue.js 3** para el monitoreo climático en tiempo real de los sectores más emblemáticos de los circuitos W y O en **Torres del Paine**, Patagonia chilena.

## ✨ Características Principales
- 🌦️ **Datos en Tiempo Real**: Conexión directa con la API de Open-Meteo para obtener pronósticos precisos por coordenadas.
- 🕒 **Sincronización de Datos**: Sistema de actualización automática programado para las 08:00 AM y 20:00 PM con persistencia en `localStorage`.
- 🎨 **Interfaz Premium**: Diseño moderno con efecto *Glassmorphism* e iconografía meteorológica personalizada mediante SVGs animados.
- 📊 **Análisis y Recomendaciones**: Cálculo de estadísticas semanales y sugerencias automáticas de equipamiento según las condiciones climáticas.
- 📱 **Totalmente Responsive**: Optimizado para dispositivos móviles y escritorio.

## 🛠️ Stack Tecnológico
- **Vue.js 3** (Composition API)
- **Vite** (Build tool)
- **Vue Router** (Sistemas de rutas dinámicas)
- **Open-Meteo API** (Suministro de datos climáticos)
- **Lucide-Vue-Next** + **Custom SVG Animations**

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/miguellucero123/portafolio_modulo6.git
   cd portafolio_modulo6
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar en desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```
