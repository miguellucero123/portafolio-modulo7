# ClimaTorre

SPA **Vue 3** + **Vue Router** + **Vuex**: clima en Torres del Paine (**Open-Meteo**), login demo, favoritos y preferencias.

## Inicio rápido

```bash
git clone https://github.com/miguellucero123/portafolio-modulo7.git
cd portafolio-modulo7
npm install
npm run dev
```

Abre **http://localhost:5174** (puerto por defecto en `vite.config.js`).

| Demo | Contraseña |
|------|------------|
| `demo@climatorre.cl` | `demo123` |

No hace falta **API key** (Open-Meteo vía `fetch`).
